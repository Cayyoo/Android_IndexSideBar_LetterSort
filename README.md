# Android_SideBar_LetterSort
侧边栏A-Z排序，通过数据库搜索城市

```java
/**
 * 共有两个列表：
 * 1、默认呈现的列表
 * 2、展示搜索内容的列表（搜索后，覆盖默认列表）
 */

/**
 * assets目录下有city.db文件
 *
 * 类似联系人列表搜索功能，通过数据库操作，查询城市。
 * 列表使用ListView
 *
 * 注意：该Demo没有加入拼音解析功能，只能以字母做为条件搜索，不能搜索汉字。
 */
```
